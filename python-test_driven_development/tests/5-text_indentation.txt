The ``5-text_indentation`` module
==================================

Using ``text_indentation``
--------------------------
>>> text_indentation = __import__('5-text_indentation').text_indentation

Standard string with mixed punctuation:
>>> text_indentation("Hello. World? This is a test: Cheers")
Hello.
<BLANKLINE>
World?
<BLANKLINE>
This is a test:
<BLANKLINE>
Cheers

String with multiple spaces after punctuation:
>>> text_indentation("End of sentence.     Next sentence starts.")
End of sentence.
<BLANKLINE>
Next sentence starts.

String with no punctuation:
>>> text_indentation("Just a normal sentence with no triggers")
Just a normal sentence with no triggers
